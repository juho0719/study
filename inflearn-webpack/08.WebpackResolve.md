# Webpack Resolve
- Webpack의 모듈 번들링 관점으로 봤을때 오듈을 어떤 위치에서 어떻게 로딩할지에 관한 정의를 하는 것이 Module Resolution
```javascript
// 일반적인 모듈 로딩 방식
import foo from 'path/to/module'
// 또는
require('path/to/module');
```
- `모듈을 어떻게 로딩해오느냐?`에 주목
1. 절대 경로를 이용한 파일 로딩
```javascript
import "/home/me/file";
import "C:\\Users\\me\\file";
```
2. 상대경로를 이용한 파일 로딩
- 해당 모듈이 로딩되는 시점의 위치에 기반하여 상대 경로를 절대 경로로 인식하여 로딩
```javascript
import "../src/file1";
import "./file2";
```
